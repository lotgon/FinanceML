Inflation adjusted price.

We want to see on chart data for decades. However, average inflation is 5-7 percent annualy. It means that 50 years ago price was 18 less. We can not compare data on one chart, so we have to ajust it by inflation. ## First approach To find best inflation coefficient, we have to find minimum of adj_price = sp500_price*((1+rate)(1/365))(.N:1) f(sp500_price, rate) = max(adj_price) - min(adj_price). Best value for sp500 is 5.3.

#next lines can be used to find best rate
#sp500[,adjustedValue:=value*((1+rate)^(1/365))^(.N:1)]
#sp500[!is.na(adjustedValue),max(adjustedValue) - min(adjustedValue)]
rate <- 0.053
sp500[,adjustedValue:=value*((1+rate)^(1/365))^(.N:1)]
ggplot(sp500)  + geom_line(aes(x=date, y=adjustedValue), color='green')+ geom_line(aes(x=date, y=value), color='blue')

Second approach

However, in real life, inflation rate is not constant yearly. Lets try another adjustment based on M2.

m2 <- fred_getSeries("M2SL", observation_start, observation_end)
stopifnot( sp500$date == m2$date )
sp500[, m2adjusted_value:=value/m2$value*m2[.N, value]]
ggplot(sp500)  + geom_line(aes(x=date, y=m2adjusted_value), color='blue')+ geom_line(aes(x=date, y=adjustedValue), color='green')

Third approach

To avoid inflation noise and normalizing, we will tranform SP500 rates to dropdawns from previous best maximum. It is standard approach. But, we will add some information about how much growth we have from last previous maximum. Local maximum will change only when dropdown > 20%.

#to observe period and magnitude of crisises we tranform price to drawdowns
sp500 <- TranformToDropdawnData(sp500)

CalculateLastMax(sp500)

ggplot(data = sp500) +
