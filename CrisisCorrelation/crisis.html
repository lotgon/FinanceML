Inflation adjusted price.

We want to see on chart data for decades. However, average inflation is 5-7 percent annualy. It means that 50 years ago price was 18 less. We can not compare data on one chart, so we have to ajust it by inflation. ## First approach To find best inflation coefficient, we have to find minimum of adj_price = sp500_price*((1+rate)(1/365))(.N:1) f(sp500_price, rate) = max(adj_price) - min(adj_price). Best value for sp500 is 5.3.

#next lines can be used to find best rate
#sp500[,adjustedValue:=value*((1+rate)^(1/365))^(.N:1)]

rate <- 0.053
sp500[,adjustedValue:=value*((1+rate)^(1/365))^(.N:1)]
ggplot(sp500)  + geom_line(aes(x=date, y=adjustedValue), color='green')+ geom_line(aes(x=date, y=value), color='blue')
